stiff_ode

stiff_ode, a MATLAB code which sets up and solves an ordinary differential equation (ODE) which exhibits stiffness, making accurate solution difficult.

Languages:

Source Code:

  • stiff_exact.m, evaluates the exact solution of the ODE.
  • stiff_deriv.m, evaluates the right hand side of the ODE.
  • stiff_parameters.m, returns parameters of the ODE.
