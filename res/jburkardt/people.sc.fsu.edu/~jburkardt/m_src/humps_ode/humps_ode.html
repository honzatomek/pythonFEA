humps_ode

humps_ode, a MATLAB code which sets up and solves an ordinary differential equation (ODE) whose solution is a double hump curve.

Languages:

humps_ode is available in a MATLAB version and an Octave version and a Python version.

Related Data and codes:

humps_ode_test

matlab_ode, MATLAB codes which set up various ordinary differential equations (ODE).

Source Code:

  • humps_deriv.m, returns the right hand side of the ODE.
  • humps_exact.m, returns the exact solution of the ODE.
  • humps_parameters.m, returns parameters of the ODE.
