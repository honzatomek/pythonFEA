kepler_ode

kepler_ode, a MATLAB code which sets up and solves the ordinary differential equations (ODE) for a Kepler two body gravitational problem.

Languages:

kepler_ode is available in a MATLAB version and an Octave version and a Python version.

Source Code:

  • kepler_conserved.m, [h1,h2]=kepler_conserved(pq) returns the conserved quantities of the ODE.
  • kepler_deriv.m, dpqdt=kepler_deriv(t,pq) returns the right hand side of the ODE.
  • kepler_parameters.m, returns parameters of the ODE.
