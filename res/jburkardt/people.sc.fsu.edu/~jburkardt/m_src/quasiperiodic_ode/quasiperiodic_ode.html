quasiperiodic_ode

quasiperiodic_ode, a MATLAB code which sets up and solves a system of ordinary differential equations (ODEs) for a problem with a quasiperiodic solution.

Languages:

quasiperiodic_ode is available in a MATLAB version and an Octave version and a Python version.

Related Data and codes:

Source Code:

  • quasiperiodic_exact.m, returns the exact solution.
  • quasiperiodic_deriv.m, evaluates the right hand side of the ODE.
  • quasiperiodic_parameters.m, returns parameters of the ODE.
