henon_heiles_ode

henon_heiles_ode, a MATLAB code which sets up and solves the Henon-Heiles ordinary differential equations (ODE) for the motion of a star moving around a galactic center.

Languages:

henon_heiles_ode is available in a MATLAB version and an Octave version and a Python version.

Source Code:

  • henon_heiles_conserved.m, evaluates conserved quantities.
  • henon_heiles_deriv.m, returns the right hand side of the ODE.
  • henon_heiles_initial.m, defines a consistent initial condition.
