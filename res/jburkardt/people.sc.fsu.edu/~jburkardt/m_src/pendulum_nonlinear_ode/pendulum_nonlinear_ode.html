pendulum_nonlinear_ode

pendulum_nonlinear_ode, a MATLAB code which sets up and solves the ordinary differential equations (ODE) that represent the behavior of a nonlinear pendulum of length L under a gravitational force of strength G.

Source Code:

  • pendulum_nonlinear_conserved.m, evaluates a quantity that should be conserved by the nonlinear pendulum ODE.
  • pendulum_nonlinear_deriv.m, evaluates the right hand side of the ODE.
  • pendulum_nonlinear_exact.m, returns the exact solution of the nonlinear pendulum ODE; however, it only returns the first component, and the result is only valid if the initial angular velocity is zero.
  • pendulum_nonlinear_parameters.m, returns parameters of the ODE.
  • pendulum_nonlinear_period.m, returns the period of the ODE.
