rkf45_test

rkf45_test, a MATLAB code which calls rkf45(), which is an RKF45 ODE solver.

Related Data and Programs:

rkf45, a MATLAB code which implements the Runge-Kutta-Fehlberg (rkf) solver for the solution of an ordinary differential equation (ODE) system.

Source Code:

  • rkf45_test.m, calls all the tests;
  • rkf45_test.sh, runs all the tests;
  • rkf45_test.txt, the output file;

  • rkf45_test04.m, solves a scalar ODE.
  • rkf45_test05.m, solves a vector ODE.
  • rkf45_test06.m, solves a scalar ODE, using one-step integration.
  • f1.m, evaluates the derivative for the ODE.
  • f2.m, evaluates the derivative for the ODE.
  • y1x.m, evaluates the exact solution of the ODE.
