quadex_ode

quadex_ode, a MATLAB code which sets up and solves a stiff ordinary differential equation (ODE), whose exact solution is a parabola, but for which errors grow exponentially.

Source Code:

  • quadex_deriv.m, returns the right hand side of the ODE.
  • quadex_exact.m, returns the exact solution of the ODE.
  • quadex_parameters.m, returns parameters of the ODE.
