COLOR
Descriptions of RGB Colors

COLOR is a dataset directory which contains information about colors in terms of RGB values.

Related Data and Programs:

COLORS, a FORTRAN90 library which converts between various formats for describing colors;

Datasets:

  • color.txt, a table listing for each color a name, a set of 32 bit integer RGB values, and a set of real RGB values.
  • color_01.txt, the RGB values for the colors, given as real numbers between 0.0 and 1.0.
  • color_255.txt, the RGB values for the colors, given as integers between 0 and 255.
  • color_names.txt, names for the colors. Currently, no name is longer than 22 characters.
