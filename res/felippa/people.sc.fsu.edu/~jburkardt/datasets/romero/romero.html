ROMERO
Latin Square => CVT Computation

ROMERO is a dataset directory which collects 12 sets of 2D Latin Square points that were used as initial generators for a CVT computation.

The LHS data was supplied by Vicente C Romero of Sandia National Laboratory.

Each of the LHS data sets contains a set of points in the unit square that were chosen by a Latin Square computation. These points were then used as initial values of the CVT cell generators for the CVT_DATASET program which computed a corresponding set of CVT points.

For the two dimensional datasets, some Encapsulated PostScript images (EPS) have been provided. These images were created by the PLOT_POINTS program.

Datasets:

  • plot_lhs.inp, the input command file for making the plots of the LHS sets;
  • plot_lhs.out, the printed output from PLOT_POINTS.
  • plot_cvt.inp, the input command file for making the plots of the CVT sets;
  • plot_cvt.out, the printed output from PLOT_POINTS.

Data set #1 is a set of 9 LHS points, using seed 1, from VCR, which are used as starting points for a CVT iteration:

  • romero_01.inp, the input commands to CVT_POINTS.
  • romero_01.out, the printed output from CVT_POINTS.
  • romero_01_lhs.txt, the LHS points.
  • romero_01_lhs_dot.png, a PNG image of the LHS dot plot.
  • romero_01_lhs_cir.png, a PNG image of the LHS circle plot.
  • romero_01_cvt.txt, the CVT points.
  • romero_01_cvt_dot.png, a PNG image of the CVT dot plot.
  • romero_01_cvt_cir.png, a PNG image of the CVT circle plot.
