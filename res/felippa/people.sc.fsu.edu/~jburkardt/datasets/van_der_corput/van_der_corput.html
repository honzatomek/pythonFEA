VAN_DER_CORPUT
van der Corput Datasets

VAN_DER_CORPUT is a dataset directory which contains points generated by the 1-dimensional van der Corput sequence.

The arguments that the user may set include:

  • BASE, the base of the computation, which might be 2, or any prime, or any integer greater than 1.
  • SEED, the index of the first element, which might be 0 or 1;
  • N, the number of elements to compute.

Related Data and Programs:

HALTON, a dataset directory which contains examples of the Halton sequence, a multi-dimensional generalization of the van der Corput sequence.

HAMMERSLEY, a dataset directory which contains examples of the Hammersley sequence, a multi-dimensional generalization of the van der Corput sequence.

SEQUENCE_STREAK_DISPLAY, a MATLAB program which can make a "streak plot" of a sequence.

TABLE, a file format which is used to store these datasets.

VAN_DER_CORPUT, a C++ library which computes elements of a 1D van der Corput quasirandom sequence.

VAN_DER_CORPUT_DATASET, a C++ program which creates a van der Corput quasirandom sequence and writes it to a file.
