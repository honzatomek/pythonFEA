FINGERPRINTS

FINGERPRINTS is a dataset directory which contains some images of fingerprints.

Licensing:

The computer code and data files described and made available on this web page are distributed under the GNU LGPL license.

Related Data and Programs:

SVD_DEMO, a MATLAB program which demonstrates the singular value decomposition (SVD) for a simple example.

SVD_FINGERPRINT, a MATLAB program which reads a file containing a fingerprint image and uses a series of low rank singular value decomposition (SVD) approximations to compress the image.
