MPS
Linear Programming Datasets

MPS is a dataset directory which contains files in MPS format which define various linear programming problems.

The data files are text files, and have a common, standard format. They tend to be large and verbose, and so it is useful to compress them when they are not in use.

Datasets:

  • adlittle.mps, defines a problem of 57 rows and 97 columns;
  • afiro.mps, defines a problem of 28 rows and 32 columns;
  • empstest.mps, a simple test file primarily useful for checking the compression and decompression programs;
  • maros.mps, an example taken from Maros, which defines a problem of 3 rows and 4 columns;
  • nazareth.mps, an example taken from Nazareth, which defines a problem of 3 rows and 3 columns;
  • testprob.mps, a simple problem with 4 rows and 3 variables;
