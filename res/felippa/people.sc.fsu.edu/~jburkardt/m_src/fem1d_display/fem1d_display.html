fem1d_display

fem1d_display, a MATLAB code which reads a prefix defining a trio of files containing the nodes, nodal values, and elements for finite element data, and displays a plot.

The finite element function is assumed to be defined by as a piecewise polynomial, which is continuous at the breakpoints, and which has the same degree within each breakpoint interval.

Usage:

fem1d_display ( 'prefix' )
where 'prefix' is the common file prefix:
  • prefix_nodes.txt lists the node coordinates;
  • prefix_values.txt lists the node values;
  • prefix_elements.txt lists the node indices that form elements;
