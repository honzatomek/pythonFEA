obj_display

obj_display, a MATLAB code which gets the name of an ASCII OBJ file from the user, reads the 3D graphics information stored in the file, and displays the structure on the screen.

Usage:

obj_display ( 'file.obj' )
where
  • 'file.obj' is the ASCII OBJ file
reads the information in the ASCII OBJ file file.obj
