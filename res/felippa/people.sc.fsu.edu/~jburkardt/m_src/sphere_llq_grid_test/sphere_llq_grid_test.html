sphere_llq_grid_test

sphere_llq_grid_test, a MATLAB code which calls sphere_llq_grid(), which uses longitudes and latitudes to create grids of points, lines, and quadrilaterals over the surface of a sphere in 3D.

sphere_llq_grid, a MATLAB code which uses longitudes and latitudes to create grids of points, lines, and quadrilaterals on the surface of the unit sphere in 3D.

Source Code:

  • sphere_llq_grid_test.m, calls all the tests;
  • sphere_llq_grid_test.sh, runs all the tests;
  • sphere_llq_grid_test.txt, the output file.

  • sphere_llq_grid.png, an image of an LLQ grid on the surface of a sphere in 3D.
