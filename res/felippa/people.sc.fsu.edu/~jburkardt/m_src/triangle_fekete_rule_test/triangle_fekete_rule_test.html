triangle_fekete_rule_test

triangle_fekete_rule_test, a MATLAB code which calls triangle_fekete_rule(), which returns information defining any of seven Fekete rules for high order interpolation and quadrature over the interior of a triangle in 2D.

Related Data and Programs:

triangle_fekete_rule, a MATLAB code which defines Fekete rules for quadrature or interpolation over the interior of a triangle in 2D.

Source Code:

  • triangle_fekete_rule_test.m, calls all the tests.
  • triangle_fekete_rule_test.sh, runs all the tests.
  • triangle_fekete_rule_test.txt, the output file.

  • triangle_fekete_rule_test01.m, tests FEKETE_RULE_NUM, FEKETE_DEGREE, and FEKETE_ORDER_NUM.
  • triangle_fekete_rule_test02.m, tests FEKETE_RULE by summing the weights.
  • triangle_fekete_rule_test03.m, tests FEKETE_RULE by summing the barycentric coordinates.
  • triangle_fekete_rule_test04.m, tests FEKETE_ORDER by integrating monomials in the unit triangle.
  • triangle_fekete_rule_test05.m, tests FEKETE_RULE by plotting the points.
  • triangle_rule_fekete_test06.m, tests REFERENCE_TO_PHYSICAL_T3 by transforming a Fekete rule from the unit triangle to another triangle.

One of the tests in the sample calling program creates EPS files of the abscissas in the unit triangle. These have been converted to PNG files for display here.
