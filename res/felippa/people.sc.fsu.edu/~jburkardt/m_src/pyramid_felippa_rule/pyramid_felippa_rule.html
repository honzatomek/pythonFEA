pyramid_felippa_rule

pyramid_felippa_rule, a MATLAB code which returns Felippa quadrature rules over the interior of the unit pyramid in 3D.

Languages:

pyramid_felippa_rule is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version.
