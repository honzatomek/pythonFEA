truncated_normal_sparse_grid_test

truncated_normal_sparse_grid_test, a MATLAB code which calls truncated_normal_sparse_grid(), which computes a sparse grid based on a normal probability density function (PDF), also called a Gaussian distribution, that has been truncated to [A,+oo), (-oo,B] or [A,B].
