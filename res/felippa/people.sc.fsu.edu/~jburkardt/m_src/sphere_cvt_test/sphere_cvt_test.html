sphere_cvt_test

sphere_cvt_test, a MATLAB code which calls sphere_cvt(), which iteratively approximates a centroidal Voronoi tessellation (CVT) on the unit sphere.

Related Data and Programs:

sphere_cvt, a MATLAB code which uses a Centroidal Voronoi Tessellation to create a mesh of well-separated points on the surface of the unit sphere in 3D.
