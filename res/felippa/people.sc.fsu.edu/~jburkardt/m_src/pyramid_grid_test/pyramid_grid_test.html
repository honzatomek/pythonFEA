pyramid_grid_test

pyramid_grid_test, a MATLAB code which calls pyramid_grid(), which generates a grid of points over the interior of a pyramid in 3D.

Source Code:

PYRAMID_UNIT is a set of data defining a GNUPLOT image of a grid on a unit pyramid.

  • pyramid_unit_commands.txt, a file of commands to be input to gnuplot.
  • pyramid_unit_nodes.txt, a file containing the coordinates of the grid points.
  • pyramid_unit_vertices.txt, a file containing the coordinates of the pyramid vertices.
  • pyramid_unit.png, an image of the grid.
