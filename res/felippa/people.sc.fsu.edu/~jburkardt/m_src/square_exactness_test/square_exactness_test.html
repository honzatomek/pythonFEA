square_exactness_test

square_exactness_test, a MATLAB code which calls square_exactness(), which investigates the polynomial exactness of quadrature rules for f(x,y) over the interior of a square (rectangle/quadrilateral) in 2D.

Source Code:

  • legendre_set.m, sets a 1D Gauss-Legendre rule.

  • padua_weight_set.m, sets the weights of a 2D Padua rule.
