legendre_rule_fast_test

legendre_rule_fast_test, a MATLAB code which calls legendre_rule_fast(), which implements a fast algorithm for the computation of the points and weights of the Gauss-Legendre quadrature rule.

Related Data and Programs:

legendre_rule_fast, a MATLAB code which uses a fast algorithm to compute a Gauss-legendre quadrature rule for estimating the integral of a function with density 1 over the interval [-1,+1].

Source Code:

  • legendre_rule_fast_test.m, calls all the tests.
  • legendre_rule_fast_test.sh, runs all the tests.
  • legendre_rule_fast_test.txt, the output file.

The following files were created by the command legendre_rule_fast ( 15, 0.0, 2.0 ):

  • leg_o15_r.txt, the region file.
  • leg_o15_w.txt, the weight file.
  • leg_o15_x.txt, the abscissa file.
