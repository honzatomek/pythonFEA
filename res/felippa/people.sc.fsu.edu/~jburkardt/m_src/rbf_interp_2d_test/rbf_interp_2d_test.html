rbf_interp_2d_test

rbf_interp_2d_test, a MATLAB code which calls rbf_interp_2d(), which defines and evaluates radial basis function (RBF) interpolants to 2D data.

Related Data and Programs:

rbf_interp_2d, a MATLAB code which defines and evaluates radial basis function (RBF) interpolants to 2D data.
