ellipse_monte_carlo

ellipse_monte_carlo, a MATLAB code which uses the Monte Carlo method to estimate the value of integrals over the interior of an ellipse in 2D.

Languages:

ellipse_monte_carlo is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.
