tetrahedron_nco_rule_test

tetrahedron_nco_rule_test, a MATLAB code which calls tetrahedron_nco_rule(), which defines the weights and abscisass for a sequence of 7 Newton-Cotes open quadrature rules over the interior of a tetrahedron in 3D.

Related Data and Programs:

tetrahedron_nco_rule, a MATLAB code which defines Newton-Cotes Open (NCO) quadrature rules over the interior of a tetrahedron in 3D.

Source Code:

  • tetrahedron_nco_test.m, calls all the tests.
  • tetrahedron_nco_test.sh, runs all the tests.
  • tetrahedron_nco_test.txt, the output file.

  • tetrahedron_nco_test01.m, tests TETRAHEDRON_NCO_RULE_NUM, TETRAHEDRON_NCO_DEGREE, and TETRAHEDRON_NCO_ORDER_NUM.
  • tetrahedron_nco_test02.m, tests TETRAHEDRON_NCO_RULE by summing the weights.
  • tetrahedron_nco_test03.m, tests TETRAHEDRON_NCO_RULE by summing the barycentric coordinates.
  • tetrahedron_nco_test04.m, tests TETRAHEDRON_NCO_ORDER by integrating monomials in the unit tetrahedron.
  • tetrahedron_nco_test05.m, tests REFERENCE_TO_PHYSICAL_T4 by transforming an NCO rule from the unit tetrahedron to another tetrahedron.
  • tetrahedron_nco_test06.m, tests TETRAHEDRON_NCO_RULE by printing out one rule.
