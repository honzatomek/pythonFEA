sphere_llq_grid

sphere_llq_grid, a MATLAB code which uses longitudes and latitudes to create grids of points, lines, and quadrilaterals over the surface of a sphere in 3D.

Languages:

sphere_llq_grid is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.
