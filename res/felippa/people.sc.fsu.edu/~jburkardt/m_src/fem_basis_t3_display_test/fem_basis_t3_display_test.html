fem_basis_t3_display_test

fem_basis_t3_display_test, a MATLAB code which calls fem_basis_t3_display(), which reads data defining a linear triangular ("T3") mesh and displays the basis function associated with any node in the mesh.

Source code

  • fem_basis_t3_display_test.m, calls all the tests.
  • fem_basis_t3_display_test.sh, runs all the tests.
  • fem_basis_t3_display_test.txt, the output file.

SQUARE is a simple mesh of 25 nodes and 32 linear triangular ("T3") elements over the [0,4] by [0,4] square:

  • square_nodes.txt, the node file.
  • square_elements.txt, the element file.
  • square_mesh.png, a PNG image of the mesh.

  • square_node2.png, a PNG image of the basis function associated with node 2.
  • square_node12.png, a PNG image of the basis function associated with node 12.
  • square_node13.png, a PNG image of the basis function associated with node 13.
