ellipsoid_monte_carlo_test

ellipsoid_monte_carlo_test, a MATLAB code which calls ellipsoid_monte_carlo(), which applies the Monte Carlo method to estimate the value of an integral over the interior of an ellipsoid in M dimensions.

ellipsoid_monte_carlo, a MATLAB code which applies a Monte Carlo method to estimate the integral of a function over the interior of an ellipsoid in 3D.

Source Code:

  • ellipsoid_monte_carlo_test03.m, tests ellipsoid_sample() on a 3D ellipsoid centered at (1,2,3).
