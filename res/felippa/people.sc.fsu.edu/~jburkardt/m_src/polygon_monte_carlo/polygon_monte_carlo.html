polygon_monte_carlo

polygon_monte_carlo, a MATLAB code which applies a Monte Carlo method to estimate the integral of a function over the interior of a polygon in 2D.

Languages:

polygon_monte_carlo is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.
