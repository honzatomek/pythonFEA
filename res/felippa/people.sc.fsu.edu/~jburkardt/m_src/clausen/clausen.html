clausen

clausen, a MATLAB code which evaluates a Chebyshev interpolant to the Clausen function Cl2(x).

Cl2(x) = - integral ( 0 < s < x ) log ( abs ( 2 sin ( x / 2 ) ) ) dx

The Clausen function is periodic, with period 2 pi.
