grid_display

grid_display, a MATLAB code which reads an ASCII file defining a set of points on a 1D, 2D or 3D grid or sparse grid, and display a plot of those points, inside a plot frame of user-specified range [Rmin,Rmax].

Usage:

grid_display ( 'file_name', rmin, rmax )
where
  • 'file_name' is the name of the file containing the data points.
  • rmin, rmax is the range in X, Y (and possibly Z) over which the plot is to be displayed. This is typically either [-1,1] or [0,1]. Some grids don't include points on the boundary, so it is useful to be able to specify the plot range, and thus force the boundaries to show up, and to show up "in the same place" for multiple plots. Also, this feature allows you to zoom in on part of the grid, if you wish.
