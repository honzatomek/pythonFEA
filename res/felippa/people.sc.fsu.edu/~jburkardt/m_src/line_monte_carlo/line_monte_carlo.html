line_monte_carlo

line_monte_carlo, a MATLAB code which uses the Monte Carlo method to estimate the integral of a function over the length of the unit line in 1D.

Languages:

line_monte_carlo is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.
