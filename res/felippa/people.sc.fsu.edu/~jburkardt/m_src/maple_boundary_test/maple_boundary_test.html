maple_boundary_test

maple_boundary_test, a MATLAB code which calls maple_boundary(), which takes an image of a maple leaf and determines the pixels that form the boundary.

Related Data and Programs:

maple_boundary, a MATLAB code which reads an image of a maple leaf and extracts the list of pixels that form the boundary.

Source Code:
