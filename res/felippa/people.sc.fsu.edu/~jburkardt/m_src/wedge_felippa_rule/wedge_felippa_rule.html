wedge_felippa_rule

wedge_felippa_rule, a MATLAB code which generates quadrature rules to estimate integrals over the interior of the unit wedge in 3D.

The interior of the unit wedge in 3D is defined by the constraints: 

        0 <= X
        0 <= Y
             X + Y <= 1
       -1 <= Z <= +1
