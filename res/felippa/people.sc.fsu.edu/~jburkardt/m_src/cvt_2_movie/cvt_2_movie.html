cvt_2_movie

cvt_2_movie, a MATLAB code which animates the generation of a Centroidal Voronoi Tessellation (CVT) in the unit square, which includes a random "rebirth" at every iteration.

Usage:

cvt_2_movie ( n, sample_num, frame_num )
where
  • n is the number of points to generate; the default is N = 100.
  • sample_num is the number of sampling points: the default is SAMPLE_NUM = 100,000.
  • frame_num is the number of frames to create. the default is FRAME_NUM = 30.
