sphere_fibonacci_grid

sphere_fibonacci_grid, a MATLAB code which constructs a grid of points using the Fibonacci spiral over the surface of a sphere in 3D.

Languages:

sphere_fibonacci_grid is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version.
