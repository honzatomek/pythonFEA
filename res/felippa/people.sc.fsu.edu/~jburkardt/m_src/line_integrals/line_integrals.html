line_integrals

line_integrals, a MATLAB code which returns the exact value of the integral of any monomial along the length of the unit monomial in 1D.

The length of the unit line in 1D is defined by 

        0 <= x <= 1

The integrands are all of the form 

        f(x) = x^e
where the exponent is a nonnegative integer.
