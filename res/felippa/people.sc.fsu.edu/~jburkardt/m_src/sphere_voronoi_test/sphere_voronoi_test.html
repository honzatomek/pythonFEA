sphere_voronoi_test

sphere_voronoi_test, a MATLAB code which calls sphere_voronoi(), which computes the Voronoi diagram of points on the unit sphere.

Related Data and Programs:

sphere_voronoi, a MATLAB code which computes the Voronoi diagram of points on the surface of the unit sphere in 3D.

Source Code:

  • sphere_voronoi_test.m, calls all the tests.
  • sphere_voronoi_test.sh, runs all the tests.
  • sphere_voronoi_test.txt, the output file.

  • sphere_voronoi_test01.m, demonstrates some of the Voronoi calculations.
  • sphere_voronoi_test02.m, displays the Voronoi diagram on a sphere.
  • sphere_voronoi_test03.m, compares naive and direct approaches to computing the area of the Voronoi polygons.
  • sphere_voronoi_test04.m, processes a set of data which is fairly regular, and generates some Voronoi vertices that are essentially equal (this used to cause problems).
  • figure_01.png, an image of the Voronoi diagram made using PATCH.
