wedge_grid

wedge_grid, a MATLAB code which generates a grid of points over the interior of the unit wedge in 3D.

The interior of the unit wedge in 3D is defined by the constraints: 

        0 <= X
        0 <= Y
             X + Y <= 1
       -1 <= Z <= +1
