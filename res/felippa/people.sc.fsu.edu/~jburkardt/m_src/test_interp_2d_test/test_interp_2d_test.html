test_interp_2d_test

test_interp_2d_test, a MATLAB code which calls test_interp_2d(), which defines test problems for interpolation of data z(x,y) depending on a 2D argument.

Related Data and Programs:

test_interp_2d, a MATLAB code which defines test problems for interpolation of regular or scattered data z(x,y), depending on a 2D argument.

Source Code:

  • test_interp_2d_test.m, calls all the tests.
  • test_interp_2d_test.sh, runs all the tests.
  • test_interp_2d_test.txt, the output file.

  • test_interp_2d_test01.m, simply prints the title of each grid and function.
  • test_interp_2d_test02.m, samples each function using each grid.
  • test_interp_2d_test03.m, plots the grid points.
