toms886_test

toms886_test, a MATLAB code which calls toms886(), which implements an interpolation procedure based on "Padua points", defined in the square [-1,+1]^2, whose interpolating power is especially good. It is possible to map these points to the general rectangle, ellipse or triangle to do interpolation on these regions as well.
