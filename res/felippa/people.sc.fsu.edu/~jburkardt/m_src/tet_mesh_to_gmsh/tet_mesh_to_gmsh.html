tet_mesh_to_gmsh

tet_mesh_to_gmsh, a MATLAB code which converts a pair of files defining a tetrahedral mesh into an equivalent file using the Gmsh msh format.

In particular, TET_MESH_TO_GMSH reads a node coordinate file listing the coordinates of a set of 3D points, and an element file list of sets of 4, 10 or 20 nodes that form tetrahedral elements. It creates a single equivalent file in the Gmsh msh format.

Usage:

tet_mesh_to_gmsh ( 'prefix' )
where
  • 'prefix'_nodes.txt contains the node definitions;
  • 'prefix'_elements.txt contains the element definitions;
  • 'prefix'.msh will be the Gmsh msh file created by the program.
