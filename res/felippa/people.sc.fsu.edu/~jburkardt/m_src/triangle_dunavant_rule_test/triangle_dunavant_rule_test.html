triangle_dunavant_rule_test

triangle_dunavant_rule_test, a MATLAB code which calls triangle_dunavant_rule(), which defines 20 quadrature rules over the interior of a triangle in 2D.

Related Data and Programs:

triangle_dunavant_rule, a MATLAB code which defines Dunavant rules for quadrature over the interior of a triangle in 2D.

Source Code:

  • triangle_dunavant_rule_test.m, calls all the tests.
  • triangle_dunavant_rule_test.sh, runs all the tests.
  • triangle_dunavant_rule_test.txt, the output file.

  • triangle_dunavant_rule_test01.m, tests DUNAVANT_ORDER_NUM.
  • triangle_dunavant_rule_test02.m, tests DUNAVANT_RULE by summing the weights.
  • triangle_dunavant_rule_test03.m, tests DUNAVANT_RULE by summing the coordinates.
  • triangle_dunavant_rule_test04.m, tests DUNAVANT_ORDER by integrating monomials in the unit triangle.
  • triangle_dunavant_rule_test05.m, tests REFERENCE_TO_PHYSICAL_T3 by transforming a rule from the unit triangle to another triangle.
