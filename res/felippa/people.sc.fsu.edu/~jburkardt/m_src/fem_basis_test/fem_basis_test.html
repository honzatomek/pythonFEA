fem_basis_test

fem_basis_test, a MATLAB code which calls fem_basis(), which evaluates basis functions associated with an M-dimensional simplex (a 1D interval, a 2D triangle, a 3D tetrahedron, and the higher-dimensional generalizations).

Related Data and Programs:

fem_basis, a MATLAB code which can define basis functions for the finite element method (FEM) for any degree in an M-dimensional simplex (1D interval, 2D triangle, 3D tetrahedron, and higher dimensional generalizations.)

Source Code:
