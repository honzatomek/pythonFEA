grids_display

grids_display, a MATLAB code which can read two ASCII files defining grids or sparse grids in 2D or 3D. The first grid is plotted "lightly", in hollow blue dots, while the second is plotted with solid red dots. The image is automatically saved to a PNG file.

Usage:

grids_display ( 'file_name1', 'file_name2', rmin, rmax )
where
  • 'file_name1' is the file containing the first grid, to be shown in hollow blue.
  • 'file_name2' is the file containing the second grid, to be shown in solid red.
  • rmin, rmax are the range in X, Y (and possibly Z) over which the plot is to be displayed. This is typically either [-1,1] or [0,1]. Some grids don't include points on the boundary, so it is useful to be able to specify the plot range, and thus force the boundaries to show up, and to show up "in the same place" for multiple plots. Also, this feature allows you to zoom in on part of the grid, if you wish.
