sphere_monte_carlo

sphere_monte_carlo, a MATLAB code which estimates the integral of F(X,Y,Z) over the surface of the unit sphere in 3D.

Licensing:

The computer code and data files described and made available on this web page are distributed under the GNU LGPL license.

Languages:

sphere_monte_carlo is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.
