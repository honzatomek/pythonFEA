laguerre_exactness_test

laguerre_exactness_test, a MATLAB code which calls laguerre_exactness(), which investigates the polynomial exactness of a Gauss-Laguerre quadrature rule for the infinite interval [0,+oo) with weight function e^(-x).

laguerre_exactness, a MATLAB code which tests the monomial exactness of quadrature rules for estimating Laguerre-type integrals of a function f(x) with density rho(x)=exp(-x) over the interval [0,+oo).
