triangle_lyness_rule_test

triangle_lyness_rule_test, a MATLAB code which calls triangle_lyness_rule(), which produces the Lyness-Jespersen family of quadrature rules over the interior of a triangle in 2D.

Related Data and Programs:

triangle_lyness_rule, a MATLAB code which returns Lyness-Jespersen quadrature rules over the interior of a triangle in 2D.

Source Code:

  • triangle_lyness_rule_test.m, calls all the tests.
  • triangle_lyness_rule_test.sh, runs all the tests.
  • triangle_lyness_rule_test.txt, the output file.

  • triangle_lyness_rule_test01.m, tests LYNESS_RULE_NUM, LYNESS_DEGREE, LYNESS_ORDER_NUM.
  • triangle_lyness_rule_test02.m, performs the weight sum test on Lyness rules.
  • triangle_lyness_rule_test03.m, performs the barycentric coordinate sum test on Lyness rules.
  • triangle_lyness_rule_test04.m, prints a rule generated by LYNESS_RULE.
  • triangle_lyness_rule_test05.m, writes a rule created by LYNESS_RULE to a file.
  • triangle_lyness_rule_test06.m, tests the Lyness rules for exact integration of monomials.
