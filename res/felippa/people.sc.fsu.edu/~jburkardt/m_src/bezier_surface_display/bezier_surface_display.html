bezier_surface_display

bezier_surface_display, a MATLAB code which gets the names of the node and rectangle files defining a Bezier surface, reads the information from the files, and displays the surface on the screen.

BEZIER_SURFACE_display was a "work in progress". I managed to read the data in and display a crude image, in which each Bezier patch is represented by a flat rectangle. Obviously, I would want to take the next step, which is to be able to sample the bicubic function and display a finer mesh of rectangles, in order to see the teapot in its glory!

Usage:

bezier_surface_display ( 'node_file', 'rectangle_file' )
where
  • 'node_file' lists the nodes;
  • 'rectangle_file' lists the indices of nodes that make up each rectangle;
