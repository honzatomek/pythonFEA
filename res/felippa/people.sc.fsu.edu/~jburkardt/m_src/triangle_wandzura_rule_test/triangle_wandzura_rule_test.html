triangle_wandzura_rule_test

triangle_wandzura_rule_test, a MATLAB code which calls triangle_wandzura_rule(), which returns any of six Wandzura rules for quadrature over the interior of a triangle in 2D.

Related Data and Programs:

triangle_wandzura_rule, a MATLAB code which returns quadrature rules of exactness 5, 10, 15, 20, 25 and 30 over the interior of the triangle in 2D.

Source Code:

  • triangle_wandzura_rule_test.m, calls all the tests.
  • triangle_wandzura_rule_test.sh, runs all the tests.
  • triangle_wandzura_rule_test.txt, the output file.

  • triangle_wandzura_rule_test01.m, tests WANDZURA_RULE_NUM, WANDZURA_DEGREE, and WANDZURA_ORDER_NUM.
  • triangle_wandzura_rule_test02.m, tests WANDZURA_RULE by summing the weights.
  • triangle_wandzura_rule_test03.m, tests WANDZURA_RULE by summing the barycentric coordinates.
  • triangle_wandzura_rule_test04.m, tests WANDZURA_ORDER by integrating monomials in the unit triangle.
  • triangle_wandzura_rule_test05.m, tests REFERENCE_TO_PHYSICAL_T3 by transforming a Wandzura rule from the unit triangle to another triangle.
