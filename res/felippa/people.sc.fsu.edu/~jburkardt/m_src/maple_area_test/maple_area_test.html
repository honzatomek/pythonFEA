maple_area_test

maple_area_test, a MATLAB code which calls maple_area(), which uses several approaches to estimate the area of a maple leaf using an image.

Related Data and Programs:

maple_area, a MATLAB code which takes the list of pixels that form the boundary of the image of a maple leaf within a picture, and uses grid, Monte Carlo, and Quasi Monte Carlo sampling to estimate the area of the leaf.

Source Code:

  • maple_area_test.m, calls all the tests.
  • maple_area_test.sh, runs all the tests.
  • maple_area_test.txt, the output file.

  • maple_area_estimate_grid_test.m
  • maple_area_estimate_mc_test.m
  • maple_area_estimate_qmc_test.m

  • maple_boundary.txt, a file containing the pixel coordinates that define the boundary of the maple leaf within the 387 x 391 pixel image.
