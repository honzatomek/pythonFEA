ball_and_stick_display_test

ball_and_stick_display_test, a MATLAB code which illustrates the use of MATLAB's 3D plotting capability to make a "ball and stick" plot.

FIRST_ORDER_DISPLAY displays a sequence of 8 images illustrating a first order method for dealing with hyperbolic equations.

  • first_order_0.png, a blank plotting field.
  • first_order_1.png, a node where we have the solution.
  • first_order_2.png, the region associated with the node.
  • first_order_3.png, the neighboring nodes.
  • first_order_4.png, fluxes are differences across the neighbors.
  • first_order_5.png, a full time step is now taken.
  • first_order_6.png, we have an estimate for the solution in this region.
  • first_order_7.png, all the nodes can be advanced in this way.

LAX_WENDROFF_1D_DISPLAY displays a sequence of images illustrating the Lax-Wendroff method in 1D.

  • lw1d00.png, a blank plotting field.
  • lw1d01.png, a node where we have the solution.
  • lw1d02.png, the region associated with the node.
  • lw1d03.png, the neighboring nodes.
  • lw1d04.png, values at midsides by averaging.
  • lw1d05.png, values at midside nodes, half time step.
  • lw1d06.png, estimate fluxes.
  • lw1d07.png, estimate derivative at node, half time step.
  • lw1d08.png, take full step to new time.
  • lw1d09.png, update all data.
  • lw1d10.png, ready for next time step.
