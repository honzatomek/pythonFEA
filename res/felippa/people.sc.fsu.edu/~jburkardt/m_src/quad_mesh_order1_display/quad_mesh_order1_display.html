quad_mesh_order1_display

quad_mesh_order1_display, a MATLAB code which can make a 3D plot of piecewise constant data defined on a quadrilateral mesh.

The input from the user includes:

  • NODE_XY(1:2,1:NODE_NUM), a list of the coordinates of nodes;
  • ELEMENT_NODE(1:4,1:ELEMENT_NUM), a list of the nodes that make up each element in the mesh;
  • VALUE(1:TRIANGLE_NUM), the value associated with each element;

Usage:

quad_mesh_order1_display ( 'prefix' )
where 'prefix' is the common file prefix:
  • 'prefix'_nodes.txt, lists the node coordinates;
  • 'prefix'_elements.txt lists the indices of nodes that form elements.
  • 'prefix'_values.txt lists the value to be assigned each element.
reads the data and makes a plot.
