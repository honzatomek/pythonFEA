triangulation_display_test

triangulation_display_test, a MATLAB code which calls triangulation_display(), which plots a triangulation.

triangulation_display, a MATLAB code which reads files defining a triangulation and displays an image.

Source Code:

BOX4 is an order 4 triangulation.

  • box4_nodes.txt, the nodes.
  • box4_nodes.png, an image of the nodes.
  • box4_elements.txt, the elements.
  • box4_elements.png, an image of the elements.
  • box4_element_neighbors.txt, the element neighbors.
  • box4_element_neighbors.png, an image of the element neighbors.

CIRCLE is a circular region, triangulated using order 3 elements.

  • circle_nodes.txt the point coordinates.
  • circle_nodes.png an image of the elements.
  • circle_elements.txt the elements
  • circle_elements.png an image of the elements.

ELBOW3 is an elbow-shaped region, triangulated using order 3 elements.

  • elbow3_nodes.txt the point coordinates.
  • elbow3_nodes.png an image of the nodes.
  • elbow3_elements.txt the elements.
  • elbow3_elements.png an image of the elements.

ELL3 is an order 3 triangulation of a set of nodes in an L-shaped region.

  • ell3_nodes.txt, the nodes.
  • ell3_nodes.png, an image of the nodes.
  • ell3_elements.txt, the elements.
  • ell3_elements.png, an image of the elements.
  • ell3_element_neighbors.txt, the element neighbor array.
  • ell3_element_neighbors.png, an image of the element neighbor labels.

GREENLAND is a triangulation of Greenland, using 33,343 nodes and 64,125 elements of order 3 elements. The image is too dense to display in full, so MATLAB's "zoom" facility was used to display a small portion of the mesh.

  • greenland_nodes.txt, the nodes.
  • greenland_elements.txt, the elements.
  • greenland_elements.png, an image of the elements, after "zooming in" twice.

HEX_HOLES3 puts a set of nodes into the double hexagonal hole region using CVT techniques, and creates an order 3 triangulation using TABLE_DELAUNAY.

  • hex_holes3_nodes.txt, the nodes.
  • hex_holes3_elements.txt, the elements.
  • hex_holes3_elements.png, an image of the elements.

IRREG6 is an order 6 triangulation.

  • irreg6_nodes.txt, the nodes.
  • irreg6_elements.txt, the elements.
  • irreg6_elements.png, an image of the elements.

LAKE3 uses constrained CVT methods to place nodes inside and on the boundary of the problem 14 triangulation region, a lake with an island. An order 3 triangulation is used.

  • lake3_nodes.txt, the nodes;
  • lake3_elements.txt, the elements;
  • lake3_elements.png, an image of the elements.
