shepard_interp_2d_test

shepard_interp_2d_test, a MATLAB code which calls shepard_interp_2d(), which defines and evaluates Shepard interpolants to 2D data, based on inverse distance weighting.

Related Data and Programs:

shepard_interp_2d, a MATLAB code which defines and evaluates Shepard interpolants to scattered 2D data, based on inverse distance weighting.

Source Code:

  • shepard_interp_2d_test.m, calls all the tests;
  • shepard_interp_2d_test.sh, runs all the tests;
  • shepard_interp_2d_test.txt, the output file.

  • shepard_interp_2d_test01.m, tests a particular set of data and a particular value of the exponent.
