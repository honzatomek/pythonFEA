hypercube_exactness_test

hypercube_exactness_test, a MATLAB code which calls hypercube_exactness(), which investigates the polynomial exactness of a quadrature rule over the unit hypercube in M dimensions.

hypercube_exactness, a MATLAB code which measures the monomial exactness of an M-dimensional quadrature rule over the interior of the unit hypercube in M dimensions.
